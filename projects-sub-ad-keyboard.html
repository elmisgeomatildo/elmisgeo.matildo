KKeys "Sakura" Mechanical Keyboard

Concept Social Media Ad

A conceptual social media advertisement for a mechanical keyboard named "Sakura". This projects explores how to balance tech product's feature with a specific visual theme, in this case, soft floral aesthetics. I created both static and motion versions to see how the design performs in different social media formats.

Role: Graphic Designer (Concept, Layout, Motion Adaptation)

Tool: Canva

The Goal:

To create an advertisement that makes the "Sakura" identity clear while ensuring the design still feels clean yet appealing to a broader audience.

Design Approach:

  • - Visual-first marketing style
  • - Supporting elements to reinforce product name and mood
  • - Clean typography hierarchy
  • - Balanced negative space
  • - Subtle feature and compatibility text to avoid clutter
  • - Adaptation into motion format for higher engagement
